# Tiny Dancer Solver
_A QOL module for hypixel skyblock rift._ 

Does the tiny dancer room for you / legit solver

## Installation 
Download fabric and fabric api for version 26.1.2 or 26.2

### Steps 
1. Download the lastest version of the mod
2. Move the jar file into your `minecraft\mods` folder
3. Restart Minecraft and enjoy!

## Usage
_The macro will start as soon as you enter the dance room_ --- Enjoy!
